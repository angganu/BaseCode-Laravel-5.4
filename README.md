## Start Project With
- Clone This Project
- Create Database 
- Rename .env.example To .env && Edit
- Compile With CMD :
```shell
 composer install
 yarn
 php artisan key:generate
 php artisan migrate
 php artisan db:seed
 npm run development
 npm run watch -- --watch-poll
 npm run production
 phpunit
 ```
